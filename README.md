## Getting Started

First, run the development server:

```bash
npm install

# development server
vercel dev
```

Open [http://localhost:3000](http://localhost:3000) with postman or in browser.

## Deploy

You need to install [vercel cli](https://vercel.com/download)

```bash
#deploy
vercel
```
